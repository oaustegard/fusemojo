# fusemojo

Mojo port of the Fuse.js bitap fuzzy-search engine (patterns up to 64 codepoints
vs Fuse.js's 32).

See `_FEATURES.md` for a top-down map of what's here — capability areas,
key symbols with line ranges, workflows, known parity gaps, and what's
deliberately scoped out. Read it first before navigating the source.
